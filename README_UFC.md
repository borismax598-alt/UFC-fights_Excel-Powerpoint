# UFC Fight Data Analysis

An exploratory data analysis of UFC fight history, covering fighters, events, locations, weight classes, and fight outcomes.

## Project Overview

This project explores UFC data to answer questions about fighter performance, fight finishes, weight class activity, and event locations. The findings are also presented in an accompanying PowerPoint presentation with charts and summaries.

## Dataset

The analysis is based on multiple related tables:

- **Fighters** – names, weight, height, date of birth
- **Fights** – fight results, winner, method of victory
- **Events** – event IDs, dates, locations
- **Locations** – cities, regions, countries
- **Fight types** – weight class per bout

## Key Questions Answered

- How many unique fighters have competed in the UFC?
- How many fights have been held in total?
- How were fights finished, and what were the finish rates?
- Which fighters have the most wins of all time?
- Who are the best strikers by KO count?
- How many fights were held in each weight class?
- Which cities hosted the most UFC events?
- What was the oldest age at which a fighter competed?

## Tools Used

- Microsoft Excel (pivot tables, charts, data cleaning)
- PowerPoint (presentation of results)

## Presentation

All findings are summarized in an accompanying PowerPoint presentation, which includes structured charts and clearly organized slides for each analytical question.
